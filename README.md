## Getting started

```
npm i

npx sequelize-cli db:create
npx sequelize-cli db:migrate
npx sequelize-cli db:seed

npm start

```
</br>

    - user login

    - all registered stock diplay

    - make trade for stock, place order for buying or selling shares
